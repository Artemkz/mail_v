# Как не попадать в спам (volkov.kz)

## Текущий статус

| Проверка | Статус |
|----------|--------|
| SPF | ✅ настроен |
| DKIM | ✅ подпись работает (`d=volkov.kz`) |
| DMARC | ✅ `p=none` |
| IP в чёрных списках | ✅ не найден |
| HELO / hostname | ✅ исправлено → `mail.volkova.kz` |
| **PTR (обратная DNS)** | ❌ **не настроен — главная причина спама** |

## 1. PTR — обязательно (у хостера VPS)

Напишите в поддержку хостинга (где арендован сервер `37.140.243.10`):

> Прошу настроить обратную DNS (PTR) для IP 37.140.243.10 на **mail.volkova.kz**

Без PTR Gmail и другие сервисы часто отправляют письма в спам.

## 2. Проверка настройки

После PTR отправьте письмо на https://www.mail-tester.com — цель **8–10/10**.

## 3. Репутация нового сервера

Новый IP и домен не имеют репутации. Первые недели:

- Не отправляйте массовые рассылки
- Начните с 5–10 писем в день
- Попросите получателей нажать «Не спам» в Gmail
- Не отправляйте ссылками-only или КАПСОМ

## 4. Как писать письма

- **От:** `Имя <artyom@volkov.kz>` (с именем, не только email)
- Тема без спам-слов: «бесплатно», «срочно», «!!!»
- Добавляйте текст в тело (не пустое письмо)
- Подпись с контактами внизу

## 5. SPF (можно ужесточить позже)

Сейчас: `v=spf1 mx a:mail.volkova.kz ip4:37.140.243.10 ~all`

Когда всё стабильно, замените `~all` на `-all` (строже).

## 6. Gmail: «Не спам»

Получатель должен:
1. Открыть папку «Спам»
2. Выбрать письмо → **«Не спам»**
3. Добавить `artyom@volkov.kz` в контакты

После нескольких таких действий Gmail запомнит отправителя.
